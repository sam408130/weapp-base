# WeApp Base Project

> A base project for wechat weapp runtime


## Clone it

```bash
$ cd path/to/root
$ git clone https://github.com/sam408130/weapp-base
$ cd [project-name]
```

## Install dependencies

```bash
$ npm install
```

## Usage

```bash
# build for production with minification
$ npm run build

# watch file changes
$ npm run watch

# generate new page
$ npm run generate

```

then, open `dist` in wechat devtools
